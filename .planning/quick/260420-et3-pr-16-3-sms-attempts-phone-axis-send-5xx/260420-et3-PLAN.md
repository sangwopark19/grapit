Infobip groupId reference (from PR #16 review):
- 1 = PENDING (accepted, will be delivered) → success
- 2 = UNDELIVERABLE → already async, not in sync response
- 3 = DELIVERED → only in delivery report, not send response
- 4 = EXPIRED → async
- **5 = REJECTED (sync rejection: invalid number, blocked sender, content rejected) → MUST be treated as 4xx**
