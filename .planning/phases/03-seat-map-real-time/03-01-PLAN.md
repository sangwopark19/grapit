    **2. Create `apps/api/src/modules/booking/booking.service.ts`:**
    - Inject: `UPSTASH_REDIS` token (Redis from @upstash/redis), `DRIZZLE` token (db), `BookingGateway`
    - `MAX_SEATS = 4` constant (per D-03)
    - `LOCK_TTL = 600` constant (10 minutes, per BOOK-03)
    - `lockSeat(userId: string, showtimeId: string, seatId: string): Promise<LockSeatResponse>`:
      1. Check user seat count: `redis.scard('user-seats:${showtimeId}:${userId}')` >= MAX_SEATS -> throw ConflictException('최대 4석까지 선택할 수 있습니다')
      2. Attempt lock: `redis.set('seat:${showtimeId}:${seatId}', userId, { nx: true, ex: LOCK_TTL })`
      3. If null -> throw ConflictException('이미 다른 사용자가 선택한 좌석입니다') with 409 status
      4. Track user seats: `redis.sadd('user-seats:${showtimeId}:${userId}', seatId)` + set TTL on the user-seats key too
      5. Broadcast: `this.gateway.broadcastSeatUpdate(showtimeId, seatId, 'locked')`
      6. Return `{ success: true, lockId: lockKey, seatId, expiresAt: Date.now() + LOCK_TTL * 1000 }`
    - `unlockSeat(userId: string, showtimeId: string, seatId: string): Promise<boolean>`:
      1. Check ownership: `redis.get('seat:${showtimeId}:${seatId}')` must equal userId
      2. If not owner -> return false
      3. Delete lock: `redis.del('seat:${showtimeId}:${seatId}')`
      4. Remove from user set: `redis.srem('user-seats:${showtimeId}:${userId}', seatId)`
      5. Broadcast: `this.gateway.broadcastSeatUpdate(showtimeId, seatId, 'released')`
      6. Return true
    - `getSeatStatus(showtimeId: string): Promise<SeatStatusResponse>`:
      1. Get locked seats from Redis: `redis.smembers('locked-seats:${showtimeId}')` (maintained by lock/unlock)
      2. Get sold seats from DB: `db.select().from(seatInventories).where(eq(seatInventories.showtimeId, showtimeId), eq(seatInventories.status, 'sold'))`
      3. Build Record<string, SeatState> merging both sources
      4. Return `{ showtimeId, seats }`
