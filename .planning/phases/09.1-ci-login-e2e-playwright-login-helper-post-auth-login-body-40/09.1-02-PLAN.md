```json
{
  "tag": "AUTH_LOGIN_DEBUG",
  "method": "POST",
  "url": "/api/v1/auth/login",
  "headers": { "content-type": "application/json", "content-length": "62", ... },

  "bodyKeys": ["email", "password"] | null,
  "bodyEmail": "admin@grapit.test" | null,
  "bodyPasswordLen": 17 | null,
  "rawBodyLen": 62 | null,
  "rawBodyUtf8": "{\"email\":\"admin@grapit.test\",\"password\":\"TestAdmin2026!\"}" | null
}
```

Decision Tree (RESEARCH.md §Architecture Patterns):
- req.body `{}` or undefined AND content-type == "application/json" → **원인 A** (Playwright body 인코딩 / body-parser 미파싱)
- req.body `{}` or undefined AND content-type 누락/다름 → **원인 B** (Content-Type 미스매치)
- req.body `{email, password}` 인데 validate 미호출 → **원인 C** (passport-local 내부)
