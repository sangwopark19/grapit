- ✓ **Product-side useRef latch** — fires once per error URL, fixes UX duplicate AND unblocks 12/12

수정 (apps/web/app/booking/[performanceId]/confirm/page.tsx):
```typescript

useEffect(() => {
  const hasError = searchParams.get('error');
  if (hasError !== 'true' || errorToastFiredRef.current) return;
  errorToastFiredRef.current = true;
  // ... rest unchanged
}, [searchParams]);
```
