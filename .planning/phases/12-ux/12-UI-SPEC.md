> **조여진 Minimalism** 방향(Linear/Vercel 계열) 아래 `globals.css @theme` 토큰을 미세 조정하고, `seat-map-viewer.tsx`에서 스테이지 방향/선택 애니메이션/미니맵/모바일 44px 터치 타겟을 해결한다. 신규 컴포넌트는 "미니맵 네비게이터" 1개에 국한하고, 나머지는 기존 토큰/컴포넌트에 비침습 변경만 허용한다.

## Spacing Scale

Reuses existing tokens in `globals.css:55-62`. 8-point grid with 4px base. **No new spacing tokens** in Phase 12 — D-01 제약.

| Token | Value | Usage in this phase |
|-------|-------|---------------------|
| xs | 4px | 미니맵 viewport rect stroke offset, 스테이지 배지 내부 icon-to-text gap |
| sm | 8px | 카드·섹션 간 좁은 간격, 홈 파일럿 미세 튜닝 시 `gap-2` |
| md | 16px | 기본 컴포넌트 간격(`p-4` / `gap-4`), 스테이지 배지 padding, 미니맵 `padding: 4px` 제외 나머지 |
| lg | 24px | 섹션 간 breathing(`mt-6` / `gap-6`), 홈 hero → 첫 섹션 간격 |
| xl | 32px | 홈 섹션 간 수직 breathing (현 `mt-12`에서 필요 시 `mt-8`로 조정 — D-04 미세 튜닝 범위) |
| 2xl | 48px | (Phase 12 미사용 — `mt-12` 유지 허용) |
| 3xl | 64px | (Phase 12 미사용) |

**예외 및 허용 절대값:**
- 미니맵 컨테이너 고정 크기: `width: 120px`, 모바일은 숨김 (D-14~D-16). 120px는 토큰 아님 — 미니맵 전용 고정 치수로 허용.
- 미니맵 절대 위치: `top: 12px; left: 12px`. Seat map container 내부 전용 offset (D-15).
- 모바일 터치 타겟 최소: 44px (WCAG 2.1 §2.5.5). 이 값은 좌석 렌더 결과이지 스페이싱 토큰이 아님.
- 좌석맵 컨테이너 높이 `min-h-[300px]`/`lg:min-h-[500px]`는 기존 값 그대로 유지.

**홈 파일럿 미세 튜닝 (D-04) 허용 범위:** 3~5줄 수준 — 아래 3가지에 한정:
1. 섹션 간 수직 리듬: 현재 `mt-12` → `mt-10` 또는 `mt-8`로 조정 가능 (lg/xl 토큰 이내).
2. hero banner와 첫 콘텐츠 섹션 사이 `pt-12` → `pt-10` 미세 조정.
3. 섹션 헤더 `mb-6` 유지(lg). 추가 여백 증감 시 `mb-4`(md)~`mb-8`(xl) 범위.

그 외 구조 변경(grid 재편, 새 컴포넌트 추가, 카드 재디자인)은 금지 — D-04 계약.

## Typography

기존 semantic 토큰(`globals.css:64-68`) + 기존 Tailwind 유틸리티 재사용. **신규 size/weight 도입 금지.** 정확히 4 sizes + 2 weights.

| Role | Size | Weight | Line Height | Where Used in Phase 12 |
|------|------|--------|-------------|------------------------|
| Display (heading) | 28px (`text-display`) | 600 (`font-semibold`) | 1.2 (`leading-[1.2]`) | 홈 섹션 타이틀(`HOT 공연` / `신규 오픈` / `장르별 바로가기`) — 기존 값 유지 |
| Heading (subsection) | 20px (`text-heading` 또는 `text-xl`) | 600 (`font-semibold`) | 1.2 | admin SVG 업로드 섹션 제목(이미 `h3 text-sm font-semibold` 이지만 Phase 12는 JSX 변경 금지, 참고용으로만 기록) |
| Body | 14px (`text-sm`) | 400 (default) | 1.5 (Tailwind default) | 스테이지 배지 라벨(`STAGE`), 좌석맵 tooltip(`text-xs` 유지), 섹션 본문 |
| Caption | 12px (`text-caption` 또는 `text-xs`) | 400 | 1.5 | 미니맵 하단 라벨(있을 경우), seat legend 가격 보조 표기 |

**Phase 12 금지:**
- 신규 폰트 스케일(`text-base`, `text-lg`, `text-2xl`, `text-3xl`) 도입 금지 — globals.css `@theme` 토큰 3개(`--text-display`/`--text-heading`/`--text-caption`) + `text-sm`/`text-xs` Tailwind 기본값만 허용.
- `font-medium` / `font-bold` 금지 — `font-semibold`(600)과 default(400)만.
- 커스텀 `leading-*` 추가 금지 — 기존 `leading-[1.2]`(heading), Tailwind default(1.5, body) 유지.

**스테이지 배지 타이포 (D-07):**
- SVG `<text>` 요소로 렌더 시: `font-size="14"`, `font-weight="600"`, `fill="var(--color-gray-600)"`.
- HTML 오버레이로 렌더 시(외부 DOM): `text-sm font-semibold text-gray-600`.

## Color

기존 `globals.css:5-46` 토큰 완전 재사용. **Brand Purple `#6C3CE0` 유지** (D-03). 60/30/10 split 준수.

| Role | Value | Token | Usage in this phase |
|------|-------|-------|---------------------|
| Dominant (60%) | `#FFFFFF` | `--color-background` / `bg-white` | 페이지 배경, 카드 표면, 좌석맵 배경(`bg-gray-50` 유지) 외 surfaces |
| Secondary (30%) | `#F5F5F7` | `--color-gray-100` / `--color-secondary` / `bg-gray-50`~`bg-gray-100` | 좌석맵 컨테이너 배경(기존 `bg-gray-50` 유지), 미니맵 컨테이너 배경(`bg-white/90` + `border-gray-200`), 스테이지 마커 fill (기존 SVG `#E5E7EB` 유지) |
| Accent (10%) | `#6C3CE0` | `--color-primary` | **Reserved-for list 아래** |
| Destructive | `#C62828` | `--color-destructive` / `--color-error` | Phase 12 에서는 직접 사용 없음. 기존 에러 fallback(`좌석 배치도를 불러오지 못했습니다`) 텍스트는 `text-gray-600` 유지 |

**Accent (`#6C3CE0`) reserved for (Phase 12):**
1. 미니맵 viewport rect `stroke` — `stroke="#6C3CE0" stroke-width="2" fill="rgba(108, 60, 224, 0.08)"` (D-14, Claude's Discretion 범위).
2. Focus ring — `--color-ring` 전역 사용 유지 (변경 없음).
3. SVG 업로드 dropzone hover 상태(`hover:border-primary hover:bg-primary/5`) — 기존 패턴 유지, 재작업 없음.
4. 좌석 선택 상태의 `fill` — 각 tier 고유 색상이 우선이며, selected stroke는 기존 `#1A1A2E` 유지(D-12 기존 로직 재활용). Purple을 selected stroke에 새로 도입하지 않음.

**Accent NOT allowed on (Phase 12):**
- 본문 텍스트, 섹션 제목, 카드 제목(기존 `text-gray-900` / `text-gray-600` 유지).
- 스테이지 배지 fill/stroke (기본 회색 계열: fill `#E5E7EB` 또는 `--color-gray-200`, text `--color-gray-600`).
- 체크마크 fade-in 애니메이션의 fill — 기존 `white` 유지 (D-12).
- 실시간 broadcast 좌석 상태 변화(locked/sold의 회색 `#D1D5DB` 유지, transition 없음 — D-13).

**Chart palette:** Phase 12 범위 밖 (어드민 대시보드만 해당, Phase 11 기배포). `--chart-1~5` 값 변경 금지.

**Shadow/Radius 토큰 추가 (D-01, Claude's Discretion):**

`@theme` 블록에 아래 토큰을 신규 추가. Tailwind v4는 `--shadow-*` / `--radius-*` 네임스페이스를 인식하여 `shadow-sm` / `rounded-lg` 등 유틸리티가 자동 반영됨.

| Token | Value | 용도 |
|-------|-------|------|
| `--radius-sm` | `6px` | Badge, tooltip |
| `--radius-md` | `8px` | Button, Input, small cards (기존 `rounded-md` 대체) |
| `--radius-lg` | `10px` | Card, dropdown, 좌석맵 컨테이너 (기존 `rounded-lg` 살짝 조정) |
| `--radius-xl` | `12px` | Dialog, 주요 컨테이너 |
| `--shadow-sm` | `0 1px 2px 0 rgba(0, 0, 0, 0.05)` | Card hover-less 기본 elevation |
| `--shadow-md` | `0 4px 12px -2px rgba(0, 0, 0, 0.08)` | Card hover, dropdown, 미니맵 패널 |

**금지:**
- `shadow-lg`/`shadow-xl`/`shadow-2xl` 신규 사용. 두 단계(`sm`/`md`)만으로 충분(Linear/Vercel convention).
- 그라디언트 배경(hero banner 포함 기존 이미지 배너는 예외).
- 네온/glow(filter: drop-shadow, blur 기반 accent glow) 효과.

**기존 하드코딩 색상(변경 없이 유지):**
- `seat-map-viewer.tsx:20-21`: `LOCKED_COLOR = '#D1D5DB'`, `SELECTED_STROKE = '#1A1A2E'` — 좌석 상태 시각 계약. Phase 12 변경 금지.
- `sample-seat-map.svg` stage fill `#E5E7EB`, stage stroke `#9CA3AF` — 기존 SVG 호환성 유지(D-08).

### Destructive actions

**Phase 12에서는 destructive action 없음.** 좌석 선택 해제는 destructive로 분류하지 않음 (즉시 복귀 가능, confirmation 불필요). SVG 교체도 `다른 SVG 업로드`로 기존 버튼 유지 — 별도 confirmation dialog 추가 없음.

### Copy tone guidelines

- 존댓말 유지 (`~하세요`, `~해주세요`).
- 영문 고유명사(`STAGE`)는 대문자 유지 (브랜딩·공연장 관습).
- 에러 카피는 "문제 + 다음 행동"을 한 문장으로 — 기존 카피 톤 유지.

## Layout Contract

### 좌석맵 뷰어 (`seat-map-viewer.tsx`)

```
┌───────────────────────────────────────────────────────┐
│ ┌─ minimap (120px, md:block hidden) ────┐             │
│ │ ┌──────┐                              │             │
│ │ │░░▣▣░░│  ← viewport rect             │             │
│ │ │░░▣▣░░│    (primary stroke 2px)      │             │
│ │ └──────┘                              │             │
│ └───────────────────────────────────────┘             │
│                                                       │
│                 [stage badge: STAGE — 상단 중앙]        │
│                                                       │
│               ┌──── seat grid SVG ────┐               │
│               │ (zoom/pan 영역)         │               │
│               │ 모바일 initialScale=1.4 │               │
│               └──────────────────────┘               │
│                                                       │
│                                       [+] zoom in     │
│                                       [−] zoom out    │
│                                       [⤢] reset       │
└───────────────────────────────────────────────────────┘
```

- **미니맵 위치 (D-15):** `absolute top-3 left-3 z-40`. 120×auto. 데스크톱(`md:` 이상)에서만 표시. 모바일은 `hidden md:block`.
- **스테이지 배지 위치 (D-07):** SVG 내부에 이미 `<text>STAGE</text>`가 있으면 뷰어는 건드리지 않음. 없고 `data-stage` 속성만 있을 경우 viewBox 기준 해당 변(top/right/bottom/left)에 SVG `<g>` 오버레이 삽입. 위치 계산은 `processedSvg` useMemo 내부에서 수행.
- **Zoom 컨트롤 (기존):** `absolute bottom-4 right-4 z-50` — 변경 없음. 미니맵 좌상단과 충돌 없음.
- **Tooltip (기존):** `pointer-events-none absolute z-50`. 미니맵보다 위 z-index(50) — 미니맵(z-40) 위에서도 가독.

### 홈 파일럿 (`app/page.tsx` + `components/home/*`)

기존 구조 유지. 허용 변경:
- `mx-auto w-full max-w-[1200px] px-4 md:px-6` (기존) 유지.
- 섹션 간 `mt-12` → 필요 시 `mt-10` 조정 (D-04, 토큰 내).
- `HotSection` / `NewSection` / `GenreGrid` 내부 JSX 변경 금지 — Swiper·grid·card 구조 유지.

### 반응형 breakpoint

- Tailwind 기본 `md` = 768px.
- `< 768px` (모바일): 미니맵 숨김(D-16), 좌석맵 `initialScale=1.4`(D-17).
- `≥ 768px` (데스크톱): 미니맵 표시, 좌석맵 `initialScale=1`.
- breakpoint 감지 방식(`window.matchMedia` 훅 vs Tailwind `md:hidden`): Claude's Discretion (planner 결정).

| **좌석 해제 (로컬)** | 선택 해제 시 체크마크 `opacity` 1→0 150ms, rect `fill` 전 색상으로 150ms. DOM 제거는 transition 완료 후 (또는 setTimeout 150ms). |
| **실시간 broadcast (locked/sold 전환)** | D-13: `transition:none` 유지. 타 관객 액션은 즉시 플립 — 혼란 방지. |
| **줌/팬 조작** | `TransformWrapper onTransformed` 콜백으로 미니맵 viewport rect의 `x`/`y`/`width`/`height`를 refs로 업데이트. React state 미사용 (성능). |

| **모바일 수동 zoom-out** | 허용 (`minScale: 0.5` 유지, D-18). 사용자가 의도적으로 줌아웃 시 44px 보장 해제됨 — "기본값 보장"이 계약. |
| **스테이지 배지 렌더** | SVG `<text>STAGE</text>` 존재 시: 뷰어는 no-op. `[data-stage]`만 존재 시: 뷰어가 viewBox 좌상단/우/하/좌 변에 `<g>` 오버레이 삽입 (SVG 파싱 시점, useMemo 내부). |
| **Admin SVG 업로드 검증 (D-06)** | 파일 업로드 직후 `FileReader`로 텍스트 추출 → `<text>STAGE</text>` 또는 `[data-stage]` 정규식/DOMParser 체크. 실패 시 toast `스테이지 마커가 없는 SVG입니다.` + 업로드 abort. |
| **`prefers-reduced-motion`** | 체크마크 fade-in / fill transition을 `@media (prefers-reduced-motion: reduce)` 내부에서 `transition: none`으로 override. CSS 기반. |
| **Focus / keyboard** | 미니맵은 시각적 요소 — Tab 순서에서 제외(`tabIndex={-1}` 또는 `aria-hidden="true"`). 좌석맵 키보드 접근성은 기존 유지 (Phase 12 범위 밖, deferred). |

## Accessibility Contract

- **좌석맵 컨테이너 (기존):** `role="grid" aria-label="좌석 배치도"` 유지.
- **스테이지 배지 (D-07):** SVG `<text>` 요소에 `aria-label="무대 위치: {top|right|bottom|left}"` 부여 (오버레이 삽입 시). 기존 SVG의 `<text>STAGE</text>`는 건드리지 않음.
- **미니맵:** `aria-label="좌석 미니맵"` 컨테이너 + viewport rect에 `aria-label="현재 보이는 영역"` (optional). 스크린리더 혼란 방지를 위해 전체 미니맵에 `aria-hidden="true"` 사용도 허용 — 좌석맵 자체가 이미 접근 가능하므로 중복 노출 방지.
- **색상 대비:**
  - 스테이지 배지 fill `#E5E7EB` + text `#6B7280` → 3:1 (기존 SVG 관습). 보조 장식 요소로 간주되어 AA grading 대상 아님. 주의: 배지 옆 좌석 번호는 기존 `#6B7280` on white = 5.7:1 (AA PASS).
  - 미니맵 viewport rect stroke `#6C3CE0` on white = 7.1:1 (AA+ PASS).
- **color-alone 금지:** 좌석 상태는 기존 tier 색상 + stroke + 체크마크(선택) 3중 시각 시그널 — color-alone 아님. 미니맵 viewport는 stroke+fill 조합.
- **터치 타겟 (D-17):** WCAG 2.1 §2.5.5 Target Size (Enhanced) — 44×44 CSS px 충족. 32px 기본 좌석 × 1.4 = 44.8px. `minScale` 0.5는 사용자 의도적 축소로 예외.
- **reduced-motion:** 선택 애니메이션 + 체크마크 fade-in은 `prefers-reduced-motion` 시 즉시 전환으로 fallback.
- **키보드 네비게이션:** Phase 12 범위 밖. 좌석맵 키보드 접근성 전면 도입은 deferred (CONTEXT §deferred 참조).
