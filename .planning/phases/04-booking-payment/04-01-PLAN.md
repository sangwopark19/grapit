    **1. ReservationService (`apps/api/src/modules/reservation/reservation.service.ts`):**

    Injectable class with `@Inject(DRIZZLE) private readonly db: DrizzleDB` and `private readonly tossClient: TossPaymentsClient`.

    Methods:
    - `generateReservationNumber(): string` — Format: `GRP-YYYYMMDD-XXXXX` where XXXXX is random alphanumeric (Math.random().toString(36).substring(2, 7).toUpperCase())
    - `calculateTotalAmount(seats: SeatSelection[], performanceId: string): Promise<number>` — Query priceTiers WHERE performanceId, match each seat.tierName to tier, sum prices. Throw BadRequestException('유효하지 않은 등급입니다') if tier not found.
    - `calculateCancelDeadline(showDateTime: Date): Date` — Returns new Date(showDateTime.getTime() - 24 * 60 * 60 * 1000)
    - `confirmAndCreateReservation(dto: ConfirmPaymentDto, userId: string): Promise<ReservationDetail>` — Steps:
      1. Query showtime by dto.showtimeId to get performanceId and dateTime
      2. Calculate expectedAmount via calculateTotalAmount
      3. If expectedAmount !== dto.amount, throw BadRequestException('금액이 일치하지 않습니다')
      4. Check idempotency: if reservation with dto.orderId already exists, return existing
      5. Call tossClient.confirmPayment({ paymentKey: dto.paymentKey, orderId: dto.orderId, amount: dto.amount })
