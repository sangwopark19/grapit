  <action>
    두 가지 CSS 버그를 동일 파일에서 수정한다.

    **Fix 1 — 포스터 이미지 잘림 (line 73, Gap 1 / cosmetic)**
    현재:
    ```
    className="relative aspect-[2/3] w-full max-h-[400px] mx-auto shrink-0 overflow-hidden rounded-lg bg-gray-200 md:max-h-none md:max-w-[380px] md:mx-0"
    ```
    수정 후:
    ```
    className="relative aspect-[2/3] w-full mx-auto shrink-0 overflow-hidden rounded-lg bg-gray-200 md:max-w-[380px] md:mx-0"
    ```
    변경 내용: `max-h-[400px]`과 `md:max-h-none` 제거.
    이유: `aspect-[2/3]`이 이미 비율(높이 = 너비 × 1.5)을 결정하므로 `max-h-[400px]`이 불필요하며, 모바일에서 컨테이너 너비가 300px 이상이면 450px 높이가 되어 400px max-h에 걸려 잘림 발생.

    **Fix 2 — 예매 버튼 MobileTabBar에 가림 (line 248, Gap 2 / major)**
    현재:
    ```
    className="fixed bottom-0 left-0 right-0 z-40 flex h-16 items-center border-t bg-white px-4 shadow-[0_-4px_6px_rgba(0,0,0,0.05)] md:hidden"
    ```
    수정 후:
    ```
    className="fixed bottom-[56px] left-0 right-0 z-40 flex h-16 items-center border-t bg-white px-4 shadow-[0_-4px_6px_rgba(0,0,0,0.05)] md:hidden"
    ```
    변경 내용: `bottom-0` → `bottom-[56px]`.
    이유: MobileTabBar는 `fixed bottom-0 z-50 h-14`(=56px). z-index도 50 vs 40으로 TabBar가 위에 렌더링되므로 CTA 버튼이 완전히 가려짐. bottom-[56px]으로 CTA 버튼을 TabBar 높이만큼 올려야 한다.
