    **Auth service (apps/api/src/modules/auth/auth.service.ts):**
    - Inject: JwtService, ConfigService, UserRepository, DRIZZLE
    - Dependencies: argon2 for password hashing, @nestjs/jwt for token signing, nodemailer for email, crypto for random bytes

    `register(dto: RegisterBody)`:
    1. Check email uniqueness (throw ConflictException if exists)
    2. Hash password with argon2.hash(password, { type: argon2id, memoryCost: 19456, timeCost: 2, parallelism: 1 })
    3. Insert user with all fields from dto
    4. Insert terms_agreement record
    5. Generate access token (JWT, 15min, payload: { sub: user.id, email: user.email, role: user.role })
    6. Generate refresh token (random 32-byte hex), hash it with SHA-256, store in refresh_tokens with family = new UUID
    7. Return { accessToken, refreshToken (raw), user: UserProfile }

    `login(user: ValidatedUser)`:
    1. Generate access + refresh token pair (same as register step 5-6)
    2. Return AuthResponse

    `validateUser(email: string, password: string)`:
    1. Find user by email
    2. If not found or no passwordHash, throw UnauthorizedException
    3. Verify with argon2.verify(user.passwordHash, password)
    4. If invalid, throw UnauthorizedException with message "이메일 또는 비밀번호가 일치하지 않습니다" (per D-08)
    5. Return user (without passwordHash)

    `refreshTokens(oldRawToken: string)`:
    1. Hash the incoming raw token with SHA-256
    2. Find refresh_tokens record by tokenHash
    3. If not found, throw UnauthorizedException
    4. If revokedAt is set: this token was already used. SECURITY: revoke ALL tokens in this family (possible token theft). Throw UnauthorizedException.
    5. If expired (expiresAt < now), throw UnauthorizedException
    6. Revoke old token (set revokedAt = now)
    7. Generate new refresh token with same family
    8. Generate new access token
    9. Return { accessToken, refreshToken (raw) }

    `revokeRefreshToken(rawToken: string)`:
    1. Hash token, find in DB, set revokedAt = now

    `requestPasswordReset(email: string)`:
    1. Find user by email. If not found, return silently (prevent email enumeration).
    2. Generate JWT reset token (payload: { sub: user.id, purpose: 'password-reset' }, expiresIn: '1h', secret: JWT_SECRET + user.passwordHash as additional entropy so token invalidates after password change)
    3. Send email via nodemailer with link: `${FRONTEND_URL}/auth/reset-password?token=${resetToken}`
    4. Return success message regardless

    `resetPassword(token: string, newPassword: string)`:
    1. Verify JWT reset token (with same compound secret)
    2. Hash new password with argon2
    3. Update user's passwordHash
    4. Revoke all refresh tokens for this user (force re-login)

    **AuthController (apps/api/src/modules/auth/auth.controller.ts):**

    @Controller('api/v1/auth')

    Endpoints:
    1. `@Public() @Post('register')` register(@Body(new ZodValidationPipe(registerBodySchema)) dto, @Res({ passthrough: true }) res):
       - Call authService.register(dto)
       - Set refresh token cookie via setRefreshTokenCookie helper
       - Return { accessToken, user }

    2. `@Public() @UseGuards(AuthGuard('local')) @Post('login')` login(@Req() req, @Res({ passthrough: true }) res):
       - User already validated by LocalAuthGuard
       - Call authService.login(req.user)
       - Set refresh token cookie
       - Return { accessToken, user }

    3. `@Public() @Post('refresh')` refresh(@Req() req, @Res({ passthrough: true }) res):
       - Extract refreshToken from req.cookies.refreshToken
       - If no token, throw UnauthorizedException
       - Call authService.refreshTokens(token)
       - Set new refresh token cookie
       - Return { accessToken }

    4. `@Post('logout')` logout(@Req() req, @Res({ passthrough: true }) res):
       - Extract refreshToken from req.cookies.refreshToken
       - If token exists, call authService.revokeRefreshToken(token)
       - Clear cookie: res.clearCookie('refreshToken', { path: '/api/v1/auth' })
       - Return { message: 'Logged out' }

    5. `@Public() @Post('password-reset/request')` requestReset(@Body(new ZodValidationPipe(resetPasswordRequestSchema)) dto):
       - Call authService.requestPasswordReset(dto.email)
       - Return { message: '비밀번호 재설정 링크를 발송했습니다' } (always, to prevent email enumeration)

    6. `@Public() @Post('password-reset/confirm')` confirmReset(@Body(new ZodValidationPipe(resetPasswordSchema)) dto):
       - Call authService.resetPassword(dto.token, dto.newPassword)
       - Return { message: '비밀번호가 변경되었습니다' }

    Private helper `setRefreshTokenCookie(res: Response, token: string)`:
    ```typescript
    res.cookie('refreshToken', token, {
      httpOnly: true,
      secure: process.env.NODE_ENV === 'production',
      sameSite: 'strict',
      maxAge: 7 * 24 * 60 * 60 * 1000, // 7 days
      path: '/api/v1/auth',
    });
    ```
