    **performances.ts:**
    - Genre enum: `pgEnum('genre', ['musical', 'concert', 'play', 'exhibition', 'classic', 'sports', 'kids_family', 'leisure_camping'])`
    - Performance status enum: `pgEnum('performance_status', ['upcoming', 'selling', 'closing_soon', 'ended'])`
    - Table `performances` with fields:
      - `id: uuid().defaultRandom().primaryKey()`
      - `title: varchar('title', { length: 255 }).notNull()`
      - `genre: genreEnum('genre').notNull()`
      - `subcategory: varchar('subcategory', { length: 100 })` — nullable, stores values like 'hot', 'original', 'domestic' etc.
      - `venueId: uuid('venue_id').references(() => venues.id)` — FK to venues
      - `posterUrl: varchar('poster_url', { length: 1000 })`
      - `description: text('description')` — rich text/HTML for detail tab
      - `startDate: timestamp('start_date', { withTimezone: true }).notNull()`
      - `endDate: timestamp('end_date', { withTimezone: true }).notNull()`
      - `runtime: varchar('runtime', { length: 50 })` — e.g. "150분 (인터미션 20분 포함)"
      - `ageRating: varchar('age_rating', { length: 50 }).notNull()` — e.g. "만 7세 이상"
      - `status: performanceStatusEnum('status').notNull().default('upcoming')`
      - `salesInfo: text('sales_info')` — cancellation/refund policy text for "판매정보" tab
      - `viewCount: integer('view_count').notNull().default(0)` — for popularity sort
      - `createdAt, updatedAt` timestamps
    - Indexes:
      - `index('idx_performances_genre').on(table.genre)`
      - `index('idx_performances_status').on(table.status)`
      - `index('idx_performances_title_trgm').using('gin', sql\`${table.title} gin_trgm_ops\`)`
