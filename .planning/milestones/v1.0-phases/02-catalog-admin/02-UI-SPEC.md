## Spacing Scale

| Token | Value | Usage |

| xl | 32px | Major section gaps, page side margins |

## Interaction Contracts

### Performance Card

| Interaction | Behavior |
|-------------|----------|
| Hover (desktop) | Card lifts with shadow-md transition (150ms ease). Poster scales to 1.02 with overflow hidden |
| Click/tap | Navigate to /performance/:id |
| Status badge | Positioned top-left of poster, offset 8px from edges |
| Poster loading | Skeleton rectangle at 2:3 ratio until image loads |

### GNB Genre Navigation

| Interaction | Behavior |
|-------------|----------|
| Genre tab click | Navigate to /genre/:genre. Active tab: primary text + 2px bottom border |
| "더보기" hover | Opens dropdown with 3 remaining genres (스포츠, 아동/가족, 레저/캠핑) |
| "더보기" dropdown | Positioned below button, 180px wide, shadow-lg, rounded-lg |
| Mobile | Genre tabs in mobile menu (hamburger). Full list, no "더보기" needed |

### Search

| Interaction | Behavior |
|-------------|----------|
| Search input focus | Ring-2 primary color. No autocomplete in Phase 2 (discretion: deferred for simplicity) |
| Enter key / search icon click | Navigate to /search?q={query}. Minimum 1 character required |
| Empty query submit | No navigation. Input shakes briefly (200ms translateX animation) |
| Clear button (X icon) | Visible when input has text. Clears input and focuses |

### Subcategory Filter Chips

| Interaction | Behavior |
|-------------|----------|
| Chip click | Updates URL searchParams (?sub={value}). Active chip: primary bg + white text. Others: gray-100 bg + gray-900 text |
| Horizontal scroll (mobile) | Chips in single row, overflow-x-auto, hide scrollbar. Snap to chip boundaries |
| "전체" chip | Selected by default. Deselects when another chip is active. Re-selects to clear filter |

### Sort Toggle

| Interaction | Behavior |
|-------------|----------|
| Toggle click | Switches between 최신순/인기순. Updates URL searchParams (?sort=latest or ?sort=popular) |
| Visual | Two-button toggle group. Active: primary text + primary bottom border. Inactive: gray-500 text |

### Ended Shows Toggle (SRCH-03)

| Interaction | Behavior |
|-------------|----------|
| Toggle | shadcn Switch component. Label "판매종료 공연 포함" left of switch |
| State | Off by default (exclude ended). On = include ended performances in results |
| URL | Updates searchParams (?ended=true or absent) |

### Pagination

| Interaction | Behavior |
|-------------|----------|
| Page number click | Navigate with searchParams (?page={n}). Scroll to top of grid |
| Current page | Primary bg + white text (36px square, rounded-lg) |
| Other pages | Gray-100 bg + gray-900 text. Hover: gray-200 bg |
| Prev/Next | Disabled when on first/last page (opacity-50, cursor-not-allowed) |
| Ellipsis | Show when total pages > 7. Pattern: 1 2 3 ... 8 9 10 |

### Admin Performance Form

| Interaction | Behavior |
|-------------|----------|
| Form validation | react-hook-form + zod. Validate on blur for each field. Error: red border + error message below field |
| Poster upload | Drag-and-drop zone with dashed border. Click to open file picker. Preview replaces zone after upload. Remove button (X) top-right of preview |
| Dynamic price rows | "등급 추가" adds a new row. Each row: text input (등급명) + number input (가격) + delete button (trash icon). Minimum 1 row |
| Showtime table | "회차 추가" opens inline row with date picker + time input. Delete button per row |
| Save button | Disabled while submitting. Shows spinner + "저장 중..." text. On success: toast "공연이 저장되었습니다" + redirect to list |
| Unsaved changes | No browser warning in Phase 2 (deferred to Phase 5 polish) |

### Admin Table

| Interaction | Behavior |
|-------------|----------|
| Row hover | Gray-50 background |
| Row click | Navigate to edit page |
| Status badge in table | Same badge component as public pages |
| Search in admin | Input above table, filters by performance name. Debounce 300ms |

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Color contrast | All text meets WCAG 2.1 AA 4.5:1 minimum. Primary #6C3CE0 on white = 5.1:1 (passes). Gray-600 #6B6B7B on white = 4.6:1 (passes) |
| Focus indicators | Ring-2 primary color on all interactive elements (buttons, inputs, links, cards). Visible on keyboard navigation |
| Landmarks | header, nav (GNB), main, aside (admin sidebar), footer on every page |
| Heading hierarchy | Each page has exactly one h1. Sections use h2. Subsections use h3. No skipping |
| Image alt text | Poster: "{performance title} 포스터". Banner: "{linked performance title} 배너" or "프로모션 배너". Cast photo: "{actor name}" |
| Pagination | nav element with aria-label="페이지 네비게이션". Current page: aria-current="page" |
| Admin table | Proper thead/tbody, th with scope="col". Row click has aria-label="공연명 수정 페이지로 이동" |
| Status badges | aria-label includes status text (e.g., "상태: 판매중") |
